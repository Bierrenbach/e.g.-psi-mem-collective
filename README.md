## 🧠 O que é Ψ(r)?

Ψ(r) é o campo funcional total que integra:
- **S(r)**: Entropia (desordem funcional)
- **I(r)**: Informação (estrutura útil)
- **β·S(r)·I(r)**: Interação entre entropia e informação

### Fórmulas exploradas:

```
Ψ(r) = η·S(r) + θ·I(r)
Ψ(r) = η·S(r) + θ·I(r) + β·S(r)·I(r)
```
